# 🌍 BFSI Global Labor Insights AI Agent

### 🚀 Live Demo: https://bfsi-global-labor-insightsgit-clylo5vyrgstur5g2sw6ba.streamlit.app/

## 📌 Project Overview
This project is a **Strategic Decision-Support Tool** designed for the Banking, Financial Services, and Insurance (BFSI) sector. It transforms macro-economic labor data into actionable executive insights using **Agentic AI**.

By leveraging the latest **Gemini 2.5/3 Flash** architecture, the application provides automated trend analysis and strategic recommendations for global C-suite executives.

## 🛠️ Key Features
* **Dynamic Data Normalization:** Automatically maps diverse CSV structures (Year, Country, Value) to ensure dashboard resiliency.
* **Executive Metrics Engine:** Real-time calculation of All-Time Highs, Lows, and Period-over-Period changes.
* **AI Strategic Reporting:** A persona-driven AI Agent that synthesizes data into a professional 3-point boardroom report.
* **Secure Credential Management:** Implemented via Streamlit Secrets to prevent API exposure and ensure production-grade security.

## 🏗️ Tech Stack
* **Language:** Python 3.10+
* **Frontend:** Streamlit (Web Deployment)
* **AI Model:** Google Gemini 2.5/3 Flash API
* **Data Science:** Pandas & Plotly Express
* **Environment:** Streamlit Cloud with Secret Management

## 🔐 Security & Deployment
This project follows professional DevSecOps practices:
1.  **Zero Hardcoding:** No API keys are stored in the source code.
2.  **Environment Variables:** Uses `st.secrets` for secure backend authentication.
3.  **Requirements Management:** Automated dependency tracking via `requirements.txt`.
